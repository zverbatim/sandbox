# Quartz Demo

## About
Print a ascii cat using the crontab styled string scheduler for quartz plugin
```
    /\___/\
   (  o o  )
   /   *   \
   \__\_/__/ meow!
     /   \
    / ___ \
    \/___\/
```

Docs and credits:
